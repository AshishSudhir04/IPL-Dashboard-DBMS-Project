🚀 TO GET STARTED (3 STEPS):

   STEP 1: Configure MySQL Password
   ─────────────────────────────────
   Edit file: backend/.env
   Change line: DB_PASSWORD=your_mysql_password
   
   STEP 2: Setup Backend (One Terminal)
   ─────────────────────────────────
   cd backend
   npm install
   npm run init-db
   npm run load-data
   npm start
   
   STEP 3: Setup Frontend (Another Terminal)
   ─────────────────────────────────
   npm install
   npm run dev
